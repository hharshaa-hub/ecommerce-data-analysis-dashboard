**# E-commerce Data Analysis Dashboard

## Overview
This project is an interactive Excel dashboard built to analyze e-commerce performance data.

## Key Features
- KPI tracking (Orders, Revenue, Ratings, Delivery Time)
- Interactive slicers for dynamic filtering
- Data visualization using charts and pivot tables

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Insights
- Faster delivery times are associated with higher customer ratings
- Revenue trends highlight key performance periods
